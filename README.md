## 共学前置

参与这个共学需要知道

1. 如何使用 github
2. 熟练使用一种开发语言, Js 最佳

## 共学模块

### 模块一

1. 类 OpenAI 的交互方式
2. 使用程序运用提示词完成 AI 接口交互

### 模块二

1. Agent 记忆功能RAG
2. Agent 工具调用，扩展函数 (和 SUI 结合)


### 模块三

1. Agent AI 工作流
2. Agent 间互相调用
